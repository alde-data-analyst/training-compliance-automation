# Training Compliance Automation

## Business Problem
Manual monitoring of training compliance required significant operational effort and time.

## Objective
Automate the identification of pending agents and improve monitoring using data analysis and dashboards.

## Tools Used
- Python
- Pandas
- Excel
- Power BI
- Data Analysis

## Solution
Developed an automated system to process training data, detect pending agents, and generate structured monitoring dashboards.

## Results
- Reduced manual workload
- Improved operational efficiency
- Better monitoring and decision-making
